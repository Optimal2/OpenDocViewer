# OpenDocViewer v2.6.0

OpenDocViewer v2.6.0 is a feature release focused on page-selection workflows, print preparation, and user-controlled zoom defaults. It is recommended for deployments that need easier curation of large document sessions before viewing or printing.

## Selection Workspace

- Added a dedicated selection workspace for managing which pages are included in the active viewer session and print output.
- Added document-aware and page-based workspace modes, including document boundary handling when reliable document metadata is available.
- Added drag-and-drop page and document ordering, rectangle selection, undo/redo, reset-to-saved behavior, and save/leave controls.
- Added source and selected-page panel modes, compact thumbnails, panel resizing, and list preview controls.
- Added full-screen page preview with keyboard navigation, add/remove actions, and visible selected/not-selected status feedback.
- Made saved selections drive the normal thumbnail pane, page navigation, and print behavior.

## Print And Toolbar

- Added reusable split toolbar buttons so an action button can keep its primary click behavior while exposing related defaults through a compact arrow menu.
- Added print default selection between active page and all pages, persisted per user.
- Updated print behavior so sessions with an active selection clearly distinguish selected pages from the full original session where relevant.
- Reworked toolbar controls used by the selection workspace to reduce wasted vertical space.

## Zoom And Viewer Preferences

- Added a custom size zoom mode with user-configurable caps for window width, window height, and actual size.
- Added a persisted default zoom/size preference for new sessions.
- Allowed custom size fields to be cleared, with blank or `0` treated as "use system/default behavior".
- Improved custom zoom labels, Swedish and English localization, and the visual consistency of the 1:1 and 1:X zoom controls.
- Kept viewer preferences persisted consistently between local storage and same-origin cookies.

## Fixes And Hardening

- Fixed selection workspace modifier-key handling so Ctrl/Shift no longer trigger unintended add-all actions.
- Fixed preview navigation so previews opened from source pages and selected pages use the correct page list.
- Improved warning markers for potentially out-of-order pages.
- Hardened zoom utility validation and documentation.
- Fixed JSDoc type syntax so documentation generation succeeds with the current JSDoc parser.

## OMP Packaging

- Updated runtime configuration documentation and site-config samples for custom size zoom settings.
- Updated the OMP component manifest during artifact-only delivery testing. OMP artifact versions remain independent from this public OpenDocViewer release version.
