# OpenDocViewer v2.6.5

OpenDocViewer v2.6.5 is a patch release for the v2.6 line. It keeps the v2.6.4 hardening baseline and focuses on runtime-configuration consistency, deployment metadata, and generated agent context files for AI-assisted tooling.

## Changes Since v2.6.4

- Hardened runtime configuration normalization for booleans, numeric bounds, default zoom modes, reset-session targets, and custom zoom factor aliases.
- Clarified JSDoc for runtime configuration helpers, including floating-point values and reset-session target fallback behavior.
- Regenerated the agent context files for AI-assisted tooling so CI validation sees the committed documentation mapping files as current for v2.6.5.
- Refined neutral public configuration samples and OMP artifact metadata without tying the public OpenDocViewer release version to the OMP component artifact version.

## Upgrade Guidance

Upgrade v2.6 deployments to v2.6.5. Earlier OpenDocViewer releases are superseded by the current runtime-configuration and documentation hardening baseline.
