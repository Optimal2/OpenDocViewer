OpenDocViewer manual

This manual is loaded from a simple HTML file in the public help folder, so deployments can replace it without rebuilding the React app.

Navigation

  • Arrow Up / Arrow Down move page by page.
  • Hold Ctrl when multiple documents are visible to move document by document.
  • Hold Shift in compare mode to target the right pane.

Selection and metadata

  • Use the Selection tab to include or exclude pages and documents.
  • Press Delete to hide the current left/main page from the saved selection. Hold Shift in compare mode to hide the right page instead.
  • Press M to open the session-wide metadata table when document metadata is available.

Printing

Open the print dialog from the toolbar and choose a print method before preparing the browser preview.
