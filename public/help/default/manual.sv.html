OpenDocViewer manual

OpenDocViewer används för att öppna, granska, jämföra, välja och skriva ut dokument i en webbläsare. Manualen beskriver standardfunktionerna. En installation kan ha egna fält, egna utskriftsorsaker, egna hjälptexter och egna regler.

Grundläggande arbetssätt

  1. Vänta tills dokumenten har laddats klart. Utskrift och vissa dokumentåtgärder är inaktiva medan laddningen pågår.
  2. Använd sidlistan med miniatyrer för att hoppa mellan sidor.
  3. Använd verktygsfältet för zoom, passning, rotation, jämförelse, urval, metadata och utskrift.
  4. Kontrollera alltid att rätt dokument och rätt sidor är valda innan du skriver ut eller sparar PDF.

Navigering

Du kan byta sida med verktygsfältets sidkontroller, miniatyrerna, musens rullhjul och tangentbordet. Sidräknaren visar aktuell sida i den laddade sessionen.

  • Föregående/nästa sida: använd pilknapparna i verktygsfältet eller tangentbordets piltangenter.
  • Page Up/Page Down: flyttar normalt en sida bakåt eller framåt.
  • Home/End: kan användas för att gå till början eller slutet om webbläsaren ger fokus till visningsytan.
  • Miniatyrer: klicka på en miniatyr för att öppna motsvarande sida.

Tangentbordsgenvägar ignoreras i textfält, listor och andra inmatningsfält så att du kan skriva utan att råka styra visaren.

Zoom och visning

Verktygsfältet innehåller vanliga zoom- och passningslägen. Exakta namn kan skilja något mellan installationer.

  • Anpassa till sida: visar hela sidan i tillgängligt utrymme.
  • Anpassa till bredd: gör sidan lika bred som visningsytan.
  • Faktisk storlek: visar sidan utan passningsskalning.
  • Zooma in/ut: ändrar visningsskalan stegvis.

Zoom påverkar hur sidan visas i OpenDocViewer. Normal utskrift bygger på utskriftsflödet, inte på exakt aktuell zoomnivå.

Miniatyrer och sidlista

Miniatyrpanelen visar sidorna i sessionen. Om flera dokument ingår kan sidlistan visa dokumentgränser eller dokumentinformation beroende på installationens konfiguration.

  • Markera en sida genom att klicka på miniatyren.
  • Använd miniatyrerna för snabb översikt över längre sessioner.
  • Vid mycket stora sessioner kan vissa miniatyrer laddas först när de behövs.

Dokument och metadata

OpenDocViewer kan visa metadata från de dokument som ingår i sessionen. Exempel kan vara patient-id, namn, enhet, dokumentdatum, sidantal eller dokument-id. Vilka metadata som finns beror på systemet som öppnar OpenDocViewer.

  • Sidmetadata: visar metadata för en viss sida eller dokumentdel.
  • Metadatamatris: visar metadata för flera dokument i samma session.
  • Utskriftsmetadata: vissa metadata kan också användas i sidhuvud och sidfot vid utskrift.

Om ett fält saknas visas det normalt inte i utskriften när mallen är konfigurerad med villkorade fält.

Urval

Om dokumenturval är aktiverat kan du välja vilka dokument eller sidor som ska ingå i arbetsflöden som stödjer urval. Urvalet påverkar exempelvis utskrift av valda sidor.

  • Kontrollera urvalet innan du skriver ut.
  • Urval kan vara dokumentbaserat eller sidbaserat beroende på installation.
  • Om inget särskilt urval används gäller normalt hela sessionen eller det val som anges i utskriftsdialogen.

Jämförelse

Jämförelseläge används för att visa två sidor eller dokumentdelar bredvid varandra. Det kan vara användbart vid manuell kontroll av versioner, kopior eller relaterade dokument.

  • Aktivera jämförelse från verktygsfältet om funktionen är tillgänglig.
  • Välj primär sida och jämförelsesida.
  • Utskrift av aktiv jämförelse kan hanteras annorlunda än utskrift av alla sidor.

Bildjusteringar och aktiv sida

Vissa installationer kan tillåta visuell justering av en aktiv sida, till exempel rotation eller annan visningsanpassning. Sådana justeringar hör normalt till aktiv-sida-flödet och bör kontrolleras noggrant före utskrift.

Utskrift av aktiv sida kan därför skilja sig från utskrift av alla sidor, eftersom aktiv sida kan behöva använda den aktuella visuella ytan i stället för originalbilden.

Utskrift

Öppna utskriftsdialogen från verktygsfältets utskriftsknapp. Dialogen kan innehålla fält för utskriftsorsak, mottagare/för vem, vattenstämpel och sidval beroende på installation.

Sidval

  • Aktiv sida: skriver ut sidan som är aktiv i visaren. Detta är bäst när aktuell sida har särskild visuell hantering.
  • Alla sidor: skriver ut hela sessionen eller hela aktuellt urval.
  • Enkelt intervall: skriver ut ett sammanhängande sidintervall.
  • Anpassade sidor: skriver ut en angiven lista, exempelvis 1,3,5-8.

Direktutskrift via HTML

Skriv ut via HTML använder webbläsarens vanliga utskriftsflöde för OpenDocViewers förberedda utskriftsvy. Webbläsarens val för stående eller liggande format gäller normalt hela utskriften.

Detta läge är enkelt och direkt, men webbläsarens förhandsgranskning kan vara långsam för många stora bildsidor.

Utskrift via PDF

Skriv ut via PDF gör att OpenDocViewer först skapar en PDF-fil av utskriftsjobbet och sedan skickar PDF-filen till webbläsarens utskrift. Under PDF-genereringen visas status för hur många sidor som är klara.

I PDF-läget kan varje sida automatiskt få stående eller liggande format baserat på sidbildens form. Det kan därför skilja sig från direktutskrift där webbläsarens formatval gäller mer generellt.

Spara PDF

Om installationen tillåter det visas knappen Spara PDF. Den skapar samma typ av PDF som PDF-utskrift, men laddar ned filen i stället för att öppna utskrift.

KOPIA-vattenstämpel

Om installationen tillåter det kan du markera Lägg till KOPIA-vattenstämpel. Då läggs en tydlig men transparent märkning på utskriften. Installationen avgör exakt text, språk och utseende.

Sidhuvud och sidfot

Utskrifter kan ha sidhuvud och sidfot. Dessa kan innehålla datum, tid, utskriftsorsak, användare, sidnummer, dokument-id och metadata. Fält som saknar värde kan döljas automatiskt om mallen är konfigurerad så.

Sidnumret i utskriftens sidhuvud är normalt sidans position i utskriftsjobbet. Dokumentets egna sidnummer kan visas separat i sidfoten.

Prestanda och stora dokument

OpenDocViewer försöker balansera prestanda och minnesanvändning. På kraftfulla datorer kan fler sidor och miniatyrer förberedas i förväg. På datorer med mindre minne kan visaren bli mer återhållsam.

  • Vänta tills laddningen är klar innan du startar stora utskrifter.
  • PDF-utskrift kan ge tydligare progress vid stora utskrifter.
  • Webbläsarens egen förhandsgranskning kan fortfarande ta tid, särskilt i Edge/Chromium.

Felsökning

  • Utskriftsknappen är inaktiv: vänta tills dokumenten har laddats klart.
  • Förhandsgranskningen är långsam: prova PDF-utskrift om funktionen finns.
  • Fel sidor skrivs ut: kontrollera sidval, intervall och urval i dialogen.
  • Metadata saknas i utskrift: kontrollera att metadata finns i sessionen och att installationens mall använder rätt fält.
  • Manualen saknar lokal information: installationen kan lägga en egen manualfil under help/site/.

Lokal anpassning

Den här standardmanualen kan ersättas per installation utan att bygga om OpenDocViewer. Lägg en lokal manual under help/site/manual.sv.html och motsvarande engelsk version under help/site/manual.en.html.
