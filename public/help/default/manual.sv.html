Manual för OpenDocViewer

Den här manualen laddas från en enkel HTML-fil i den publika help-mappen, så att varje installation kan byta innehåll utan att bygga om React-appen.

Navigering

  • Pil upp / Pil ned flyttar sida för sida.
  • Håll Ctrl när flera dokument är synliga för att flytta dokument för dokument.
  • Håll Shift i jämförelseläge för att styra högersidan.

Urval och metadata

  • Använd fliken Urval för att inkludera eller exkludera sidor och dokument.
  • Tryck Delete för att dölja aktuell vänster-/huvudsida från det sparade urvalet. Håll Shift i jämförelseläge för att dölja högersidan i stället.
  • Tryck M för att öppna metadata-tabellen för hela sessionen när dokumentmetadata finns.

Utskrift

Öppna utskriftsdialogen från verktygsfältet och välj utskriftsmetod innan webbläsaren förbereder förhandsgranskningen.
